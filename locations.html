Locations.jsx

for After Effects

creates 2D and 3D geo located markers on an equirectangular map or projected onto a sphere.

{% capture mddisloc %} This script was created and published in the end of 2012. While doing a motion graphics job for the city of [Bremerhaven](http://bremerhaven.de/) we had to display some cities that where close by on a 3D sphere. We could not just guess where these locations might be. It had to be acurrate, like on a real map. Fortunatly I already had wirtten AEMap and some experiments creating geo marker in After Effects. The hardest part was parsing .csv datasets. After the job was done I took the time to write an user interface, record a tutorial and create a demo video for my script so other people could use it as well. It is published on [aescipts.com](http://aescripts.com/locations) and open source under MIT license. {% endcapture %} {{ mddisloc | markdownify }}
