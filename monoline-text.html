Monoline Text.jsx

for After Effects

creates text from a single stroke vector set as mask on solids, shape layers or motionpaths. You can have all on one single layer, separate it by line or have every character as an individual layer.

{% capture mddisloc %} At the end of 2012 I have been playing a lot with the After Effects plugin Plexus. The option to represent text with it was intriguing. While trying to find a font that only consists out of one single stroke I stumbled open a script by Jongware that created single stroke text in Illustrator. He used a character set Dr. A. V. Hershey developd around 1967 initially ment for engraving. This set is in the public domain and seemed to me like a good excercise decoding data. Until now I only reorganized the data provided by Jongware who ported the simplex font from C. So there is still a lot to do with it. It is published on [aescipts.com](http://aescripts.com/monoline-text) and open source under MIT license. {% endcapture %} {{ mddisloc | markdownify }}
