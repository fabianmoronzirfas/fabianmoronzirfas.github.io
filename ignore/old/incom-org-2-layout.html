Incom.org 2 Layout

Was an attempt to fully automate a book.

{% capture mddistypo %} For my second colloquium I had to create a print documentation of all projects I did. I already had lots of [documentations](https://incom.org/profil/270) on [incom.org](https://about.incom.org/). Thanks to [Marcus Paeschke](http://mphasize.de/) I got access to an unofficial API. That gave me the possibility to import all content via json. The result does not work out of the box. I even think without my support nobody else can use it but still… This set of scripts creates from a large dataset some nice graphics and a book. It is published on [github.com](https://github.com/fabiantheblind/incomOrg2Layout) and under open source license. {% endcapture %} {{ mddistypo | markdownify }}
